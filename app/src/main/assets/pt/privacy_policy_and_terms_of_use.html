Política de Privacidade

Com a nossa aplicação Smart4Health eTranslation, o utilizador pode traduzir automaticamente diferentes documentos com dados de saúde alojados nas plataformas conectadas. A proteção dos seus dados pessoais é muito importante para nós. Estas informações descrevem a forma como a aplicação Smart4Health eTranslation trata os dados pessoais. As secções 1 a 3 informam sobre os direitos legais e as secções 4 a 6 explicam como os dados pessoais são tratados pela aplicação Smart4Health eTranslation.

1. Recolha e tratamento de dados pessoais

1.1. A aplicação Smart4Health eTranslation só é armazenada informação no dispositivo pessoal e a sua utilização não implica qualquer recolha de dados pessoais.

1.2 Todos os dados recolhidos são armazenados apenas no telemóvel pessoal. Caso o utilizador pretenda enviar e armazenar os dados fora do seu telemóvel pessoal, deve consultar os Termos de Uso e a polícia de privacidade dos respetivos sistemas externos.

1.3 No caso do pedido de assistencia por e-mail, precisamos do endereço de e-mail para fornecer uma resposta. Só utilizaremos estas informações, bem como a data e hora do contacto, para processar o pedido. Os dados não serão transmitidos a terceiros, mas apenas utilizados internamente pelo departamento responsável pela resposta à questão.

2. Direitos do utilizador

2.1 O utilizador pode pedir, a qualquer momento, para aceder, retificar, apagar, restringir o processamento ou receber os seus dados pessoais em formato portátil, bem como se opor ao seu tratamento.

2.2 O utilizador pode solicitar a confirmação de que os seus dados pessoais estão sendo processados. Neste caso, fornecemos uma cópia dos dados que estão a ser processados.

2.3. O utilizador tem o direito de retirar o seu consentimento para o processamento de dados nos termos mencionados anteriormente. Isto não afeta o processamento anteriormente realizado.

2.4 Para exercer os seus direitos o utilizador deve enviar um e-mail para helpdesk@smart4health.eu

3. Finalidade e base jurídica do tratamento dos dados pessoais

a. Armazenamento local e processamento de dados (técnicos) e informação no dispositivo individual

A aplicação Smart4Health eTranslation requer apenas um mínimo de dados para garantir o serviço e as funcionalidades da aplicação Smart4Health eTranslation. Na utilização da aplicação Smart4Health eTranslation os seguintes dados são processados e persistem apenas localmente, o que significa no dispositivo do utilizador (por exemplo, smartphone, computador ou tablet):

  • -Informação sobre as credenciais técnicas para aceder aos dados de saúde, por exemplo, a partir da plataforma de dados de saúde do cidadão do projeto Smart4Health 

  • -As configurações do utilizador 

Para manter a quantidade de dados pessoais tão limitada quanto possível, a aplicação Smart4Health eTranslation não exige um registo pelo utilizador na aplicação. Em vez disso, cria uma conta anônima temporária usando o serviço de terceiros Google Firebase Authentication para permitir a comunicação necessária e segura para os servidores. Para este efeito, o endereço IP do utilizador é temporariamente armazenado pelo Google. O Firebase é uma plataforma desenvolvida pelo Google para criar aplicações móveis e web. A Google fornece os seus serviços em relação à plataforma Firebase como processador de dados para o HPI.

O tratamento destes dados é tecnicamente necessário e é realizado para fornecer os serviços ao utilizador. A base jurídica para o tratamento acima referido é o artigo 6, n.º 1, parágrafo 1, alínea b), do RGPD (o tratamento é necessário para a execução de um contrato no qual o titular dos dados é parte).

b. Utilização de sistemas externos, como a plataforma de dados de saúde do cidadão do projeto Smart4Health e o serviço eTranslation da Comissão Europeia

Quando o utilizador se conecta a sistemas externos está acedendo a serviços de terceiros. Esta ligação é atualmente possível para a plataforma Smart4Health, os seus termos e condições, e a política de privacidade pode ser consultada em https://app.smart4health.eu/legal/terms.

Quando o utilizador solicita traduções, tem acesso ao serviço de terceiros fornecido pela Comissão Europeia e aceita a sua Política de Privacidade e Termos de Utilização.

C. Suporte/Contato

Quando o utilizador entra em contato usando o e-mail de contato fornecido na aplicação Smart4Health eTranslation, os dados que fornece (por exemplo, endereço de e-mail e nome, bem como o conteúdo da questão) serão armazenados para processar e responder às perguntas ou preocupações ou para fornecer o suporte necessário para a utilização da aplicação Smart4Health eTranslation.

Os dados que surgem neste contexto são excluídos após o armazenamento deixar de ser necessário (geralmente depois da questão ser completamente responsida) ou restringido o processamento se existirem obrigações legais de armazenamento. Dependendo do conteúdo do pedido, a base jurídica para o tratamento descrito acima é o artigo 6º, n.º 1, parágrafo 1, alínea f) do RGPD (o tratamento é necessário para salvaguardar os interesses legítimos do responsável pelo tratamento) ou o artigo 6º, n.º 1, parágrafo 1, alínea b) do RGPD (o tratamento é necessário para a execução de um contrato com o titular dos dados) se o utilizador já tiver uma conta e tiver dúvidas sobre a utilização da aplicação Smart4Health eTranslation. O objetivo do processamento de dados é processar a questão do utilizador, o que constitui um interesse legítimo no tratamento dos dados.

d. Obrigação de fornecer dados pessoais

Se o utilizador deseja entrar em contato, deve fornecer os dados pessoais necessários para o processamento e resposta à pergunta. Se o utilizador não fornecer esses dados, não é possível processar e responder à solicitação. Os dados recolhidos neste contexto são apagados após o armazenamento deixar de ser necessário ou restringido o processamento se houver obrigações legais de armazenamento.

A base jurídica é o artigo 6º, nº 1, parágrafo 1, alíneas b) e f) do RGPD. O objetivo do processamento de dados é processar a questão, o que constitui um interesse legítimo no tratamento dos dados.

e. Análise anônima para fins de solução de problemas

É utilizado o serviço de terceiros Google Firebase na aplicação para a análise de possíveis erros na aplicação eTranslation e, portanto, para a otimização contínua da aplicação. O Google, na sua função de processador de dados, processa os seguintes dados anónimos para esta finalidade:

  • Dados técnicos, tais como informações sobre o dispositivo utilizado e o  

  • Tempo de um possível erro ou falha do sistema 

  • IDs de instalação do Firebase / ID aleatório 

Os dados acima mencionados são utilizados exclusivamente para identificar o respetivo erro ou falha e para corrigir tais problemas. Nem nós nem o Google podem tirar qualquer conclusão sobre o utilizador com base nos dados recolhidos para a resolução de problemas. Firebase armazena os dados acima mencionados por um período máximo de 90 dias.

A base jurídica para o processamento acima mencionado é o nosso interesse legítimo (artigo 6º, nº 1, parágrafo 1, alinea f) RGPD) para ser capaz de identificar rapidamente possíveis fontes de erros e causas para falhas da aplicação, de forma a ser capaz de garantir um uso estável e seguro da aplicação para todos os utilizadores.
